# sarok1 DXF Viewer

Web viewer for the CNC panel DXF files generated by the svg2dxf pipeline.

## Run locally

```bash
npm install   # installs deps and patches dxf-viewer (postinstall)
npm run dev   # builds bundle, serves on localhost
```

Subsequent runs only need `npm run dev`.

## Build

```bash
npm run build
```

Outputs `dxf.bundle.js`, copies fonts, zips all DXF files into `sarok1.zip`.
